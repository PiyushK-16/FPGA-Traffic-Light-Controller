# FPGA-Based Traffic Light Controller Using Verilog HDL

## Abstract

Traffic signal control is an important application of digital systems used to regulate vehicle movement at road intersections. This project implements a Traffic Light Controller using Verilog HDL and Finite State Machine (FSM) design methodology. The controller manages traffic flow between North-South and East-West roads through a predefined sequence of traffic light states. The design is verified through simulation using EDA Playground and waveform analysis. The project demonstrates key VLSI concepts including FSM design, sequential logic, combinational logic, counters, and digital verification.

---

# 1. Introduction

Traffic lights are essential components of modern transportation systems. They help regulate vehicle movement, reduce accidents, and improve traffic efficiency. Traditionally, traffic light systems are controlled using embedded controllers and digital logic circuits.

In this project, a Traffic Light Controller is designed using Verilog HDL and implemented as a Finite State Machine (FSM). The controller cycles through multiple states representing different traffic light conditions for two intersecting roads.

The project serves as a practical example of FPGA-based digital system design and demonstrates how hardware description languages can be used to develop real-world control systems.

# 3. Problem Statement

At a road intersection, traffic from multiple directions must be controlled safely and efficiently. If both directions are allowed to move simultaneously, accidents may occur.

The objective is to develop a digital controller that:

* Allows one direction to move while stopping the other.
* Provides warning through yellow signals.
* Cycles continuously through predefined traffic states.
* Maintains safe traffic operation.

# 5. Digital Design Concepts Used

## FPGA

An FPGA (Field Programmable Gate Array) is a programmable hardware device used to implement digital circuits. It allows designers to create custom logic without manufacturing an ASIC.

## Verilog HDL

Verilog is a Hardware Description Language used to describe digital systems at the Register Transfer Level (RTL).

## Finite State Machine (FSM)

FSM is a mathematical model used to represent systems that operate through a sequence of states and transitions.

## Sequential Logic

Sequential logic stores system state information using flip-flops and changes state based on clock events.

## Combinational Logic

Combinational logic generates outputs based on current inputs and current FSM state.

## Timer Counter

A counter is used to maintain each traffic signal state for a specified duration.

## Testbench

The testbench provides simulation stimulus and verifies design functionality.

## Waveform Analysis

Waveforms help verify correct timing and state transitions.

# 6. FSM Design

The Traffic Light Controller uses four states.

## State S0

North-South Road : Green

East-West Road : Red

Duration : 10 Clock Cycles

## State S1

North-South Road : Yellow

East-West Road : Red

Duration : 3 Clock Cycles

## State S2

North-South Road : Red

East-West Road : Green

Duration : 10 Clock Cycles

## State S3

North-South Road : Red

East-West Road : Yellow

Duration : 3 Clock Cycles

FSM Sequence:

S0 → S1 → S2 → S3 → S0

# 9. RTL Design Description

The RTL design consists of:

* State Register
* Timer Counter
* State Transition Logic
* Output Logic

The state register stores the current traffic signal state.

The timer counter determines how long the FSM remains in a particular state.

The transition logic moves the FSM to the next state once the timer reaches its predefined value.

The output logic generates traffic signal outputs based on the current state.

# 12. Simulation Results

The design was simulated using EDA Playground.

The waveform confirms:

* FSM starts in State S0 after reset.
* State transitions occur correctly.
* Traffic outputs match state definitions.
* No illegal traffic combinations occur.
* Green signals are never active simultaneously for both directions.

Observed Sequence:

S0 → S1 → S2 → S3 → S0

The waveform validates successful operation of the Traffic Light Controller.
